# Taruto

A online borad game, can played without your money.

## Install

```sh
git clone https://github.com/zhixuan2333/taruto.git
cd taruto
# install client dependencies
cd ./client
yarn install
# install server dependencies
cd ../server
yarn install
```

## Run

```sh
cd ./client
yarn start
```

Run server in another terminal

```sh
cd ./server
yarn start
```
